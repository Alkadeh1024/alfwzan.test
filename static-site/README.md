# مركز الفوزان لتحفيظ القرآن الكريم

موقع ثابت (Static) قابل للنشر مباشرة على GitHub Pages بدون أي خادم.

## كيف تنشر الموقع على GitHub Pages

1. أنشئ مستودعاً جديداً على GitHub (مثلاً: `alfwzan-site`).
2. ارفع كامل محتوى هذا المجلد (`index.html`, `page1.html`, `page2.html`, `page3.html`, `admin.html`, `css/`, `js/`, `data/`, `assets/`, `.nojekyll`).
3. من المستودع → **Settings → Pages** → اختر فرع `main` ومجلد `/ (root)` ثم احفظ.
4. سيُنشر الموقع على رابط من نوع: `https://<username>.github.io/<repo-name>/`.

## بيانات الدخول

- **المشرف الأساسي** (تعديل كامل): `alfwzan@admin` / `Alfwzan1900`
- **حساب الشيخ** (تحضير وتقدير الحفظ فقط): `alfwzan@msh` / `Alfwzan1999`

افتح صفحة `admin.html` لتسجيل الدخول.

## كيف تُحفظ التغييرات

- جميع التعديلات تُحفظ في **المتصفح** (localStorage) فور قيامك بها، حتى تختبرها مباشرة.
- لكي تظهر التغييرات لكل الزوّار على GitHub Pages، اضغط في لوحة المشرف على **«تصدير site.json»** لتنزيل الملف، ثم استبدِل ملف `data/site.json` في المستودع بهذا الملف الجديد، واعمل commit & push. سيتحدث الموقع تلقائياً.
- يمكنك أيضاً تحرير `data/site.json` مباشرةً على GitHub من واجهة الويب.

## الصفحات

- `index.html` — الصفحة الرئيسية (لمحة عنا + التسجيل + معرض صور وفيديوهات).
- `page1.html` — الإنجازات (صور وفيديوهات مع وصف، يمكن التكبير بالضغط).
- `page2.html` — الأنشطة (الحالية والقادمة + عدّاد تنازلي).
- `page3.html` — بحث الطلاب (بالاسم، الرقم، أو اسم الحلقة).
- `admin.html` — لوحة المشرف.

## الشعار

من لوحة المشرف الأساسي → **الإعدادات والشعار** يمكن رفع صورة شعار جديدة في أي وقت، تظهر فوراً في أعلى يمين كل الصفحات.

## ملاحظات
- الصور تُخزَّن داخل `site.json` بصيغة Base64 (تُضغط تلقائياً عند الرفع).
- للفيديوهات الكبيرة، يُفضَّل لصق رابط فيديو خارجي (YouTube مثلاً) بدلاً من رفع الملف.
- ملف `.nojekyll` مهم لمنع GitHub من معالجة الملفات بـ Jekyll.
