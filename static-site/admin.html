تحميل البيانات الأولية

يستبدل كل البيانات المحلية (الحلقات والطلاب والإنجازات والأنشطة...) بالبيانات الموجودة في ملف data/site.json الموجود على السيرفر. مفيد عند أول إعداد أو لإعادة تعيين البيانات.

إعدادات النشر على GitHub

حتى يُحفَظ كل ما تضيفه (طلاب، صور، حلقات...) داخل المستودع تلقائياً ولا يختفي إذا عدّلت ملفات الموقع. تحتاج إنشاء رمز شخصي (Personal Access Token) من صفحة GitHub Tokens بصلاحية Contents: Read & Write على مستودعك فقط.

ملاحظة: الرمز يُحفظ محلياً في متصفحك فقط (localStorage على هذا الجهاز). لا يُرسل لأي طرف غير GitHub مباشرة. يُنصَح باستخدام Fine-grained PAT بصلاحية Contents: Read & Write على مستودع الموقع فقط، وإلغائه فوراً عند الحاجة.
